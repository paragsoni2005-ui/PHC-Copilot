## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| space-1 | 4px | Icon gaps, tight padding |
| space-2 | 8px | Compact element spacing (buttons, inputs) |

| space-6 | 24px | Layout spacing, margin-md, page margin |
| space-8 | 32px | Section gaps |
| space-12 | 48px | Large layout sections |

Exceptions:
- sidebar-width: 280px (desktop navigation rail width)
- gutter: 16px (grid gap between columns)

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#f7f9fb` | Background base (Level 0) |
| Secondary (30%) | `#ffffff` | Card surfaces, sidebar, inputs (Level 1) |
| Accent (10%) | `#091426` | Deep Slate (Primary CTA, main text) |
| AI Accent | `#0d9488` | Clinical Teal (AI action buttons, highlights) |
| AI Glow | `#e0f2fe` | Translucent background, soft glowing borders |
| Destructive | `#ba1a1a` | Status error, red warning timeline alerts |
| Warning | `#f59e0b` | Yellow/orange warning timeline alerts |
| Success | `#10b981` | Green success timeline alerts |
| Outline | `#e2e8f0` | Subtle borders for containers |

Accent reserved for:
- Primary slate (`#091426`): Main header title, primary CTA buttons (e.g. Save, Next).
- Clinical teal (`#0d9488`): "Generate Briefing" button, AI Sparkles logo, AI suggestion cards, active suggestion headers.

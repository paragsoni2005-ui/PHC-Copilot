# UI Design Specification (ui.md)

# PHC Copilot
**AI-Powered Operations Assistant for Primary Health Centres**

---

# Design Goal

Design a modern, clean, healthcare-focused dashboard inspired by Google Material 3 with a professional government-healthcare aesthetic.

The interface should feel:
- Intelligent
- Minimal
- Professional
- Data-driven
- AI-first
- Easy to understand even for non-technical healthcare workers

Avoid a traditional Hospital Management System look.

Instead, create the feel of an **AI Operations Command Center**.
